# Chapter 02: JS Concepts

This folder introduces the fundamental ways to declare variables in JavaScript.

## Files

### 05_JS_Basics.js
A quick overview of the three variable declaration keywords in JavaScript:
- `var` - function-scoped, can be redeclared and reassigned
- `let` - block-scoped, can be reassigned but not redeclared in the same scope
- `const` - block-scoped, cannot be reassigned or redeclared

The file demonstrates declaring variables with all three keywords and reassigning `var`.

## Key Takeaway
Understanding when to use `var`, `let`, and `const` is critical for writing bug-free JavaScript, especially in automation scripts.

## How to Run
```bash
node 05_JS_Basics.js
```
