## Tips

- **Keybinding Conflicts:** If a shortcut does not work, check if another extension or application has captured it.
- **Customize Shortcuts:** Open Keyboard Shortcuts editor with `Ctrl + K Ctrl + S` to search, view, and modify any shortcut.
- **Contextual Shortcuts:** Some shortcuts change behavior depending on the current context (e.g., terminal focused vs. editor focused).
